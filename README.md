# rff-data-gov-guidance
Repo for Resources for the Future's Data Governance Working Group's guidance on data science practices for RFF researchers

# Instructions for Previewing Quarto Site
1. Install Quarto

    a. Download the Quarto CLI for Windows via [this link, ](https://quarto.org/docs/get-started/)
   
    b. Double click the `quarto-x.x.xx-win.msi` file.

    c. Follow the install steps

2. Open a terminal like git bash
3. Navigate to your repository clone (or clone the repository via the `git clone` command) in the terminal
4. Enter the command:
```
quarto preview
```
This should render the website, and should open a browser window with the website.
